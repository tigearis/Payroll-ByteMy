### Permission Levels
- **Developer**: Full system access including development tools
- **Org Admin**: Full access except development tools
- **Manager**: Team oversight, payroll approval, client management
- **Consultant**: Payroll processing, assigned clients, basic reporting
- **Viewer**: Read-only access to limited information
